Редактор ip-сокетов СУДОС Цирконий-М

Программа для создания или редактирования уже имеющихся ip-сокетов (dbf, fpt) СУДОС Цирконий-М.

О проекте

При работе с ip-сокетами СУДОС Цирконий-М достаточно проблематично не ошибиться в редактировании записей сокетов, особенно при добавлении новых АРМов на уже действующей БД. Для автоматизации создания и редактирования сокетов создана данная программа.

Основные функции

Кроссплатформенность

Возможность работы программы на хостах с ОС Windows7-11x64/AstraLinux1.7.

Максимальная простота в работе

Доступ к записям сокетов организован в виде таблицы, которая доступна для редактирования.

Портабальность

Состав программы - бинарный файл. Эталон сокетов встроен в бинарник.

Используемые технологии

Golang Fyne Dbase-go

После старта и при отсутствие в папке с программой файлов socket.dbf|socket.fpt создаются два пустых файла сокетов на основе эталонов, встроенных в бинарник, либо открываются уже имеющиеся файлы сокетов для редактирования. При редактировании существующих записей (изменить ip, имя...) достаточно в нужном поле просто изменить текст и после перевода фокуса на другое поле изменение автоматически сохраняется в файл. Для редактирования файлов типа dbf|fpt Foxpro использовался модуль обработки таких файлов с прямым побайтным смещением чтением/записью.
