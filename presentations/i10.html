Просмотровщик протокола СУДОС

Просмотровщик протоколов низкого уровня (i10/!!!Trace_pu/Common_log) СУДОС Цирконий-М.

О проекте

Существует 3 вида текстовых протокола низкого уровня (текст построчный с hex-пакетами) в СУДОС Цирконий-М. Для получения статистики, расшифровки и поиска нужного пакета в протоколе требуется много ручной работы административного персонала.

Внешний вид приложения

Основные функции

Максимальная простота в работе

Доступ к 3 типам протоколов СУДОС с их автоматическим форматированием для вывода пользователю.

Портабальность

Состав программы - бинарный файл. Используется встроеный .net4.0 фреймворк.

Используемые технологии

Windows XP-10 C# WinForms SQLite3

Программа позволяет загрузить в память (таблица SQLite) строки из выбранного протокола. Тип и формат протокола (их 3 типа) анализируется в фоне и по завершении выводится в таблице формы в объёме нескольких страниц. Далее при навигации выполняется плавная загрузка необходимых страниц из таблицы SQLite, в результате чего возможно рпаботать с протоколами объёмом более 500Мб, что не доступно большенству текстовых редакторов! Реализована фильтрация сообщений по лучам, устройствам и командам периферии, вырезки по интервалу времени. Имеется возможность подсветки поп-ап окна с расшифровкой события выбранной строки. Удобный функционал - онлайн чтение протокола: новые пакеты автоматически появляются в форме программы по мере их поступления.
